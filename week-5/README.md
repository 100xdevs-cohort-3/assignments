# Taskify

Create a Full-Stack Task Management Application.

### Note: Try to do it by yourself.

### Setup Guide

#### Backend Setup
1. go inside week-5/backend and run:

```
 npm install 
```

2. copy .env.example to .env.
```
cp .env.example .env
```

3. run the server.
```
npm run dev
```
start building.
